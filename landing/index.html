Four pillars

Privacy as infrastructure, not a feature.

Most browsers add private mode as a button. Ram makes private the only mode — and shows you exactly what it's doing.

01 · Wipe

Every 24 hours, the slate is clean.

Cookies, cache, history, IndexedDB, service workers, localStorage, downloads — all of it purged on a fixed cycle. Tabs survive via encrypted snapshot; nothing else does.

How it works

Three minutes to a clean machine.

No accounts, no logins, no cloud sync — ever. Install runs locally, encrypts a profile to disk, brings up the tunnel, and starts the clock.

1install

Download. Open. Done.

Notarised binary, signed by us. SHA-256 published next to every release. No installer that phones home — ever.

2profile

Pick a 6-digit PIN.

Argon2id-derived key encrypts every byte of state on disk. We never see it. Lose it and the profile is gone — that's the design.

3browse

The clock starts.

VPN comes up, kill-switch arms, the wipe timer begins. You browse normally. At T-zero, the slate clears itself.
